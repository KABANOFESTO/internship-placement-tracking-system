# 🚀 InternTrack – Online Internship Placement & Tracking System

InternTrack is a full-stack web application designed to streamline the management of student internships. It supports internship applications, placement, supervision, progress tracking, and evaluation within academic and healthcare institutions.

---

## 📌 Features

### 👤 User Management
- Role-based authentication (Student, Supervisor, Coordinator, Admin)
- Secure login and registration
- Profile management

### 📝 Internship Applications
- Apply for internships online
- Upload CVs and supporting documents
- Track application status

### 🤖 Placement & Recommendation
- Skill-based internship matching
- Supervisor assignment
- Placement workflow management

### 📊 Progress Tracking
- Task and activity logging
- Weekly timesheets
- Milestone tracking

### 🧑‍🏫 Supervisor Evaluation
- Mid-term and final evaluations
- Performance rating and feedback
- Attendance tracking

### 📄 Report Submission
- Upload internship reports
- Supervisor feedback and resubmission
- Submission history tracking

### 🔔 Notifications
- Real-time alerts and reminders
- Messaging between users

### 📁 Document Management
- Centralized document storage
- Secure file access and sharing

### 📈 Analytics & Reporting
- Internship performance insights
- Placement statistics
- Custom reports
